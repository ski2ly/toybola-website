## 🎨 Как используется фон

Фон интегрирован **дизайнерским способом**:

### 1. Параллакс эффект
```css
bg-fixed /* Фиксированный фон при прокрутке */
```

### 2. Градиентные оверлеи
- **Сверху вниз:** `from-blue-900/90 via-blue-800/80 to-transparent`
- **Снизу вверх:** `from-gray-50 via-transparent to-transparent`

Это обеспечивает:
- ✅ Читаемость текста
- ✅ Плавный переход к контенту
- ✅ Глубину и объём

### 3. Декоративные элементы
- **Красное пятно:** `top-20 right-20` (анимированное)
- **Синее пятно:** `bottom-40 left-20` (анимированное с задержкой)

### 4. Волна снизу
SVG разделитель между Hero секцией и следующим блоком.

## 🔄 Если нужно изменить

### Заменить логотип
1. Положите новый файл в `frontend/public/images/logo/`
2. Назовите `logotoybola_main.svg` (или обновите имя в коде)
3. Frontend автоматически подхватит изменения

### Заменить фон
1. Положите новый файл в `frontend/public/images/backgrounds/`
2. Назовите `background.svg` (или обновите имя в коде)
3. При необходимости настройте градиенты:
   ```vue
   bg-gradient-to-r from-blue-900/90 via-blue-800/80 to-transparent
   ```

### Настроить размеры логотипа
Измените классы в компонентах:
- `h-10` → высота в хедере
- `h-20 md:h-28 lg:h-32` → высота на главном экране
- `h-10` → высота в футере

## 💡 Дизайнерские приёмы

### 1. Drop Shadow для логотипа
```html
class="drop-shadow-2xl"
```
Создаёт красивую тень для логотипа на цветном фоне.

### 2. Hover эффект для логотипа
```html
class="transform hover:scale-105 transition-transform duration-300"
```
Логотип увеличивается при наведении.

### 3. Анимированные пятна
```html
class="animate-pulse" style="animation-delay: 1s;"
```
Создают "живой" эффект фона.

### 4. Градиентный текст
```vue
<span class="text-brand-yellow">производителя</span>
```
Выделяет ключевые слова.
