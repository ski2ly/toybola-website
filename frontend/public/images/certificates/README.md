# 📜 Сертификаты

## Загрузка сертификатов

Поместите файлы сертификатов в эту папку:

**Путь:** `frontend/public/images/certificates/`

### Формат именования:
- `certificate-1.jpg`
- `certificate-2.jpg`
- `iso-9001.jpg`
- `en71.jpg`
- и т.д.

### Требования:
- **Формат:** JPG или PNG
- **Размер:** до 2 MB каждый файл
- **Ориентация:** любая (вертикальная или горизонтальная)
- **Качество:** высокое для отображения деталей

---

## 📸 Отображение

Сертификаты будут отображены:
- В разделе "О компании"
- В виде галереи с возможностью просмотра
- С подписями и описанием
