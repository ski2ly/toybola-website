## 🔄 Добавление новых сертификатов

### Шаг 1: Подготовьте файл

**Требования:**
- Формат: PNG или JPG
- Размер: до 5 MB
- Качество: высокое (300 DPI для печати)
- Ориентация: любая

### Шаг 2: Сохраните в папку

Поместите файл в папку:
```
frontend/public/images/certificates/
```

### Шаг 3: Обновите код

Откройте файл `frontend/src/pages/About.vue`

Найдите массив `certificates` и добавьте новый элемент:

```javascript
const certificates = ref([
  { id: 1, name: 'Сертификат соответствия', image: '/images/certificates/certificate_1.png' },
  { id: 2, name: 'ISO 9001', image: '/images/certificates/certificate_2.png' },
  { id: 3, name: 'EN71', image: '/images/certificates/certificate_3.png' },
  { id: 4, name: 'ASTM F963', image: '/images/certificates/certificate_4.png' },
  { id: 5, name: 'Сертификат ЕАЭС', image: '/images/certificates/certificate_5.png' },
  // Добавьте новую строку:
  { id: 6, name: 'Название сертификата', image: '/images/certificates/ваш-файл.png' },
])
```

### Шаг 4: Проверьте результат

Откройте сайт: http://localhost:5173/about

## 🖼️ Рекомендации по изображениям

### Для веба:
- **Формат:** PNG (для сертификатов с текстом)
- **Размер:** 1200x1600 px (вертикально)
- **Качество:** 80-90%
- **Размер файла:** < 500 KB

### Для печати:
- **Формат:** PNG или TIFF
- **Размер:** оригинальный скан
- **Качество:** 300 DPI
- **Размер файла:** до 5 MB
