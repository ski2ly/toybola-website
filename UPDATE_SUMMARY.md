### 2. Добавлены профессиональные анимации

#### CSS анимации (`main.css`):
- `fadeInUp` - плавное появление снизу
- `fadeInDown` - плавное появление сверху
- `fadeInLeft` - появление слева
- `fadeInRight` - появление справа
- `pulse` - медленная пульсация
- `slowSpin` - медленное вращение
- `scaleIn` - плавное увеличение
- `mapPulse` - пульсация для карты
- `listAppear` - появление списка

#### Классы анимаций:
- `.animate-fade-in-up`
- `.animate-fade-in-down`
- `.animate-fade-in-left`
- `.animate-fade-in-right`
- `.animate-pulse-slow`
- `.animate-spin-slow`
- `.animate-scale-in`
- `.animate-map-pulse`

#### Задержки:

## 🚀 Как использовать

### 1. Загрузка сертификатов

Поместите файлы в:
```
frontend/public/images/certificates/
```

**Формат:**
- `certificate-1.jpg`
- `certificate-2.jpg`
- `iso-9001.jpg`
- и т.д.

### 2. Настройка карты

Для анимированной карты экспорта создайте отдельный компонент или используйте готовую секцию в `About.vue`.

### 3. Изменение анимаций

Все анимации в `main.css`. Пример:
```css
.animate-fade-in-up {
  animation: fadeInUp 0.8s ease-out forwards;
}
```
